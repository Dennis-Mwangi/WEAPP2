
 Features

- **User Authentication**: Sign up, log in, and log out functionality with JWT tokens.
- **Weather Search**: Search weather by city or use geolocation.
- **Favorites**: Save favorite cities for quick access.
- **Recent Searches**: Keep track of last searched cities.
- **7-Day Forecast**: View detailed forecasts for selected cities.
- **Dynamic UI**: Glassmorphism cards, animated fade-in, smooth transitions.
- **Theme Toggle**: Switch between light and dark modes.
- **Responsive Design**: Optimized for desktop and mobile devices.



 Tech Stack

- **Frontend**: HTML5, CSS3, TailwindCSS, JavaScript  
- **Backend**: FAST API endpoints (`/weather` and `/forecast`) with JWT authentication  
- **Storage**: LocalStorage for favorites and recent searches
